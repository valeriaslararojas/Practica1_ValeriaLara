### ✨ Descripción

Esta práctica consiste en el **modelado y simulación de un sistema de EDOs**, utilizando Matlab y Simulink..

El objetivo es representar matemáticamente el sistema de un pendulo invertido y observar su respuesta mediante simulaciones, las cuales representan el socilamiento ante la friccion.
